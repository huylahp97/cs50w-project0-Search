Search by image

Search Google with an image instead of text. Try dragging an image here.

How to get an image URL

  1. Right click an image on the web and copy the image's URL.
  2. Return to the search box and paste on the URL (Ctrl+V or Command+V).

Tip: Try dragging an image into the search box from your computer or the web

How to upload an image
